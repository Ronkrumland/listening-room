# Roadmap

## Current Focus

- Build the minimal ambient now-playing display.

## Planned Features

### Now Playing Display

- Show album art, title, artist, and album.
- Show paused state.
- Handle no-track state gracefully.

### Playback Controls

- Play / pause
- Next / previous
- Controls fade out when inactive

### Display Modes

- Basic player
- Record player
- Cassette player

## Deferred

- WebSockets
- Theme system
- Playlist builder integration

## Explicitly Out of Scope For Now

- Multi-user support
- Spotify content caching
- Social/sharing features
